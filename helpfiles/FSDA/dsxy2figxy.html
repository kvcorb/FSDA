dsxy2figxy

Transform point or position from data space coordinates into normalized figure coordinates (that is coordinates with values between 0 and 1).

Syntax

[figx figy] = dsxy2figxy([x1 y1],[x2 y2])
figpos = dsxy2figxy([x1 y1 width height])


[figx figy] = dsxy2figxy(axes_handle, [x1 y1],[x2 y2])
figpos = dsxy2figxy(axes_handle, [x1 y1 width height])

Description

[figx figy] = dsxy2figxy([x1 y1],[x2 y2]) Transforms the two points (x1 y1) and (x2 y2) from data space coordinates into the normalized (that is between 0 and 1) figure coordinates (figx(1) figx(2)) and (figy(1) figy(2)).

This function has been written by MathWorks, Inc. but it is not a standard MATLAB function. However, the function is extensively documented in the help page Example: Pinning Text Arrows and Ellipses (follow the link or type dsxy2figxy in the help search).

The graphical functions in the FSDA toolbox make extensive use of dsxy2figxy.
