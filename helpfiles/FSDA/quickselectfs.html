quickselectFS

Finds the k-th ordered statistic.

Syntax

y = quickselectFS(A,k)
y = quickselectFS(A,k,kiniindex)

Description

y = quickselectFS(A,k) returns the element in A that is larger than exactly k - 1 other elements of A.

y = quickselectFS(A,k,kiniindex) produces the same output but uses kiniindex in order to speed up computations. kiniindex is the index of an element in A that is supposed to be "close" to the desired k-th order statistic.

References :

Blum M, Floyd RW, Pratt VR, Rivest RL, Tarjan RE (1973). “Time Bounds for Selection.” J. Comput. Syst. Sci., 7, 448–461.

Floyd RW, Rivest RL (1975). “Expected time bounds for selection.” Commun. ACM, 18, 165–172.

Press WH, Teukolski SA, Vetterling WT, Flannery BP (1992). Numerical Recipes in C, 2nd Edition. Cambridge University Press. ISBN 0-521-43108-5.

Example

Take a random sample A, (without replacement) of n integers from the intergers 1:n. In this case the k-th order statistic of A is equal to k.

n=50;
seq=(1:n)';
A=randsampleFS(n,n);
orderstat=zeros(n,1);
for i=1:n
    orderstat(i)=quickselectFS(A,i);
end
disp(isequal(seq,orderstat))
