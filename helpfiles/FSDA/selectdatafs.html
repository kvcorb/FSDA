selectdataFS

Graphical selection of data points using the mouse, on scatter plot matrices or trajectories in the forward plots.

Syntax

pointslist = selectdataFS
pointslist=selectdataFS(param1,val1,param2,val2,...)
[pointslist,xselect,yselect] = selectdataFS(param1,val1,param2,val2,...)

Description

This function is an extension of the selectdata function by John D'Errico that can be found in the Mathworks' page http://www.mathworks.com/matlabcentral/fileexchange/13857. Our extension is used in the FSDA toolbox to provide the forward plots and other specific graphical functions with brushing and linking functionalities. Using this toolbox generally the user does not call function selectdataFS directly, but indirectly inside functions resfwdplot, fanplot, mdrplot, yXplot, resindexplot, or malindexplot, through options databrush. For using selectdata in other more general contexts we suggest using the original version of John D'Errico and referring to the native documentation in the m-file.

pointslist = selectdataFS allows the user to select data points on a given plot and return their coordinates into pointslist.

pointslist = selectdataFS enables the user to select points with the mouse using default 'lasso' mode and all the other default options and to have row numbers in vector pointslist.

pointslist=selectdataFS(param1,val1,param2,val2,...) enables the user to select points in several ways and to have row numbers in vector pointslist.

[pointslist,xselect,yselect] = selectdataFS(param1,val1,param2,val2,...) enables the user to select points in several ways and to have row numbers in vector pointslist, x coordinates in vector xselect and y coordinates in vector yselect.

This is a list of selectdataFS properties:

Parameter Value
'Action'

character containing 'list' or 'delete'.
If Action=list (default) selected points are not deleted.
If Action=delete selected points are deleted from the plot.

'Axes'

Select the axes from which to select points. The default is to select the axes of current figure. (that is 'gca').
'BrushShape'

Sets the shape of the brush to be used. Both brush shapes are relative to the figure axes, so a nominally "circular" brush is actually elliptical if the axis units/lengths are unequal.
Remark: this option is only used when SelectionMode is 'brush'
'BrushSize'

The default value will specify a rectangular brush that has dimensions of 5% of the current axes. Notice that on a set of square axes, the brush will always look square, even if the axes have very different units.
Remark 1: Only used when SelectionMode is 'brush'.
Remark 2: 0 < brushsize <= 0.25
'FlagSize';

Size of the points which are brushed. Default is 5.
FlagSize must be >0
'Identify'

character containing 'on' or 'off'.
If Identify=on (default) selected points are temporarily over-plotted with a new filled colored marker ('o') as they are selected..
If Identify=off selected points are not over-plotted with a new filled colored marker .

Remark: points selected with a lasso, or rect may be deselected as the tool is modified. Brush selections are cumulative.
'Ignore'

A list of data handles to be ignored. This allows to use selectdata on only some sets of points, while others in the same figure are ignored. This is a useful option when you may have plotted some data points but also a curve fit through your data or a confidence band. You can then cause the plotted curve or the confidence lines to be ignored by selectdata. The default is [], that is every line (curve) in the plot can be selected.
'Label'

Character containing 'on' or 'off'.
If Label=on (default), text labels with their (x,y) coordinates appear next to each selected point.
If Label=off, text labels with coordinates do not appear next to each selected point.

Remark: points selected with a lasso, or rect may be deselected as the tool is modified. Brush selections are cumulative.
'Pointer'

Character which changes the cursor pointer while selection is active. After selection terminates, the figure pointer is restored to its old setting. The possible values of pointer are the following: {'crosshair' | 'fullcrosshair' | 'arrow' | 'ibeam' | 'watch' | 'topl' | 'topr' | 'botl' | 'botr' | 'left' | 'top' | 'right' | 'bottom' | 'circle' | 'cross' | 'fleur' | 'hand' }. The default value is 'crosshair'
'Tag'

String which identifies the plot in which the user has to select the data. In case of repeated call to function selectdataFS the user may wrongly select data from different plot. If a string is supplied in option 'Tag' every time the user tries to select points in a plot different from the one with the particular specified Tag, the procedure restarts.
'Return'

Character containing 'selected' or 'unselected'.
If Return=selected (default) selection of data points returns that set of points selected.
If Return=unselected selection of data points returns that set of points not selected.
'SelectionMode'

Character containing 'Lasso' or 'Brush' or 'Rect' or 'Closest'. 
If SelectionMode=Lasso , then the brush will be a polygon with a size as defined by the 'BrushSize' property. The brush will be centered at the current mouse coordinates. Brush size will be a fraction of the axis limits for the current axes. Click inside the axes, then drag the "brush" around. Any points the brush crosses over will be selected.
If SelectionMode=Brush then the brush will be an oval with a size as defined by the 'BrushSize' property. The brush will be centered at the current mouse coordinates. Brush size will be a fraction of the axis limits for the current axes. Click inside the axes, then drag the "brush" around. Any points the brush crosses over will be selected.
If SelectionMode=Rect then the brush will be rectangular. It is possible to click inside the axes to define one corner of the rect, and drag to specify the opposite corner, just as rbbox would do.
If SelectionMode=Closest a single mouse click in the figure window is used, then that point which is closest in in Euclidean distance (in window units) is chosen. It is possible to move the cursor around (don't release the mouse until you are done) and the currently selected point will be highlighted.

'Verify'

Character containing 'on' or 'off'.
If Verify=on  a dialog box pops up after selection. The user can then accept the selection, redo it, or cancel out, causing no points to be selected.
If Verify=off (default) no dialog box appears.
If cancel is chosen from the dialog, and 'return' was specify to return those points 'unselected', then ALL the points will actually be returned.

'FlagColor'

Character which specifies the color of the brushed points.
The default is to use a red color that is 'r'
Remark: FlagColor must be a valid color spec.

'FlagMarker'

Character which specifies the marker used to brush points.
The default is to use a red color that is symbol type 'o'
Remark: FlagMarker must be a valid plot marker.

'RemoveTool'

Character containing 'on' or 'off'.
If RemoveTool=on  yellow tool after selection is removed.
If RemoveTool=off (default) yellow tool after selection is not removed

'RemoveFlagged'

Character containing 'on' or 'off'.
If RemoveFlagged=on  filled red color for selected points after selection is removed.
If RemoveTool=off (default) filled red color for selected points after selection is not removed
