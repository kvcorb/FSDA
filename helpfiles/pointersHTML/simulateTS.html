Description

simulateLM simulates linear regression data. It is possible to specify: 1) the requested value of R2 (or equivaletly its SNR); 2) the values of the beta coefficients (possibly sparse); 3) the correlation (covariance) matrix among the explanatory variables. 4) the value of the intercept term. 5) the distribution to use to generate the Xs; 6) the distribution to use to generate the ys. 7) the MSOM contamination in Xs and ys. 8) the VIOM contamination in ys.
