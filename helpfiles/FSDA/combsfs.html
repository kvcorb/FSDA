combsFS

Creates a matrix which contains, in the rows, all possible combinations of n elements taken k at time.

Syntax

combsFS(n,k)
[out] = combsFS(n,k)

Description

This function is a sub-function of nchoosek from the Statistics Toolbox. It was extracted from nchoosek and included in the FSDA toolbox for performance reasons. For applications in more general contexts we suggest referring to the standard nchoosek MATLAB function.

    Note   This function is only practical for situations where M is less than about 15.

Example

>> v = [1 2 3 4 5];

>> combsFS(v,3)

ans =

1 2 3
1 2 4
1 2 5
1 3 4
1 3 5
1 4 5
2 3 4
2 3 5
2 4 5
3 4 5
