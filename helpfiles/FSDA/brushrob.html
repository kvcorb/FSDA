brushROB

Interactive brushing starting from the index plot of robust residuals

Syntax

brushROB

Description

brushROB opens a graphical user interface where it is possible to select different datasets and brush the residuals from the 'index plot of robust residuals' produced with LTS, LMS, S estimators or MM estimators.
