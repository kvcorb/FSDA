rescale

Rescales data in a given interval

Syntax

y = rescale(x)
y = rescale(x,a)
y = rescale(x,a,b)

Description

y = rescale(x) rescales data in matrix x inside interval [0 1]. The minimum value of A will become 0 and the maximum value of A will become 1. All the other values of A will be transformed as follows: (x-min(A))/(max(A)-min(A)).

y = rescale(x,a) rescales data in matrix x inside interval [0 a] if a>0 or inside interval [a,0] if a<0

y = rescale(x,a,b) rescales data in matrix x inside interval [a,b]  if b>a or inside interval [b,a] if a>b

y = rescale(x) returns inside y a matrix of rescaled values in the required interval

Note   rescale(x,0,1) is equivalent to rescale(x).

Examples

rescale([20 25 40])
% produces
%  0    0.2500    1.0000
