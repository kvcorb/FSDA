You can start to familiarize with the FSDA toolbox in one of the following ways.

A. Read file getting_started.mlx located in subfolder doc of the main folder of FSDA
B. Run the examples contained in files examples_regression.m or examples_multivariate.m or examples_categorical.m 
   Notice that all examples are organized in cells
C. Run the tutorials in the FSDA Matlab help pages 
D. Watch the videos in the Examples section of the FSDA Matlab help pages 
E. Read section "Introduction to robust statistics" or 
	"Technical introduction to Robust Statistics" in the FSDA Matlab help pages 


For advanced developers
The full repository of the FSDA project is in github at the web address: https://github.com/UniprJRC
