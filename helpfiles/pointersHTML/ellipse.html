ellipse

ellipse Generates an ellipse given mu (location vector) and Sigma (scatter matrix)

Description

The ellipse is generated using the equation: \[ (x-\mu)' \Sigma^{-1} (x-\mu) = c_{conflev}^2. \] The length of the i-th principal semiaxis $(i=1, 2)$ is $c \lambda_i$ where $\lambda_i$ is an eigenvalue of $\Sigma$.
