shuffling

shuffling Does a random permutation of the elements of input vector

Description

More About

If set $x$ has $t$ elements, the objective is to obtain each of the $t!$ pemutations with equal probability, especially when $t$ is large. To achieve this goal, we use backward Knuth's shuffling, which is actually based on the Fisher-Yates shuffle. Once compiled, Knuth solution is more efficient than the natural MATLAB solution x(randperm(numel(x)).

References

Knuth, Donald E. (1969). The Art of Computer Programming volume 2, Seminumerical algorithms, Reading, MA: Addison-Wesley, pp. 124-125.
