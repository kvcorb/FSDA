Clustering Data Sets

File Description of Data Set
cluster3 The dataset has been introduced by the Valladolid group.
geyser The geyser dataset is taken from the MASS library (Venables,W. N. and Ripley, B. D. (2002)). It contains information on 272 successive eruptions of the old faithful geyser in Yellowstone National Park, Wyoming. The variables are: (y1) the duration of the ith eruption, (y2) the waiting time to the start of that eruption from the start of eruption i-1.
M5data The dataset is proposed by Garcia-Escudero et al. (2008) for assessing some trimming-based robust clustering methods. The data are obtained from three normal bi variate distributions with fixed centers but different scales and proportions. One of the components is very overlapped with another one. A 10% background noise is added uniformly distributed in a rectangle containing the three mixture components, but without overlapping much with them. The third column contains the component id.
mixture100 The dataset has been introduced by the Valladolid group.
structurednoise The dataset has been introduced by the Valladolid group.
