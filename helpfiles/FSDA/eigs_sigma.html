eigs_sigma

eigs_sigma sets a non zero value for the optional parameter sigma of function eigs. Function eigs, when sigma is omitted, returns the eigenvalues largest in magnitude; when sigma is zero, the eigenvalues smallest in magnitude aree found. However, when there are duplicated rows in the input matrix, option sigma=0 produces an error which can be circumvented by changing zero into an appropriately small positive value. Given input data y, function eigs_sigma choses such a positive sigma value.
