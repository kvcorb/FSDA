mcdCorAna

Computes Minimum Covariance Determinant in correspondence analysis

More About

MCDcorAna computes the MCD estimator for a contingency table. This estimator is given by the subset of s Profile rows with smallest covariance determinant. The MCD location estimate is then the mean of those h Profile points. The default value of h is roughly 0.5n (where n is the total number of observations), but the user may choose each value between n/2 and n.

References

Greenacre, M.J. (1993), "Correspondence Analysis in Practice", London, Academic Press.
