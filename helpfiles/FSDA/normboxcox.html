normBoxCox

Normalized Box Cox transformation

Syntax

[Ytra] = normBoxCox(Y,[],la)
[Ytra] = normBoxCox(Y,ColToTra,la)

Description

[Ytra] = normBoxCox(Y,[],la) transforms all columns of matrix Y using the values specified in vector la.

[Ytra] = normBoxCox(Y,ColToTra,la) transforms the columns of Y specified in vector ColToTra using the values specified in vector la.

Examples

Example 1

Example of transformation of all the variables 
load('mussels.mat');
Y=mussels.data;
% Specify vector which contains values of transformation parameters
la=[1/3 1/3 1/3 0 0];
% Trasform using third root the first 3 variables and using log for the remaining two variables
[Ytra] = normBoxCox(Y,[],la)

 

Example 2

Example of transformation of a selected number of variables

In this second example we transform variables 2 and 5 using respectively the square root and the log transformation. 

load('mussels.mat');
Y=mussels.data;
% Specify vector which contains values of transformation parameters
la=[0.5 0];
% Transform using square root for variable 2 and log for variable 5
[Ytra] = normBoxCox(Y,[2,5],la)
