FSRbonfbound

Bonferroni bounds of minimum deletion residual along the forward search

Syntax

[out] = FSRbonfbound(n,p)
[out] = FSRbonfbound(Y,param1,val1,param2,val2,...)

Bbound = FSRbonfbound(n,p) computes the Bonferroni theoretical bounds of the minimum deletion residual for quantiles 1%, 50% and 99%.

n : scalar, number of observations

p : number of variables

  • Bbound − (n-m0+1)-by-(length(prob)+1)
    1st col = fwd search index from m0 to n-1
    2nd col = Bonferroni bound  for quantile prob[1]
    3rd col = Bonferroni bound for quantile prob[2]
    ...
    (k+1)th col = Bonferroni bound for quantile prob[k]

Bbound  = FSRbonfbound(n,p,val1,param2,val2,...) specifies one or more of the name/value pairs described in the following table.

Parameter Value
'init'

Scalar which specifies the initial subset size to compute the Bonferroni bound for minimum deletion residual, if init is not specified it will be set equal to init=floor(0.5*(n+p+1))+1.
'prob'

1-by-k vector containing quantiles for which Bonferroni bounds  have to be computed. The default is to produce 1%, 50% and 99% envelopes.

Examples

Example 1

The purpose of this example is to compare the Bonferroni bounds with those which are found using the theory of order statistics. 
MDRenv = FSRenvmdr(1000,5,'exact',1,'init',10,'prob',[0.01 0.5 0.99 0.999]);
Bbound = FSRbonfbound(1000,5,'init',10,'prob',[0.01 0.5 0.99 0.999]);
plot(MDRenv(:,1),MDRenv(:,2:5),Bbound(:,1),Bbound(:,2:5));
The two curves are very different except for the last step.
