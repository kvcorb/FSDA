shuffling

A random permutation of the elements of a vector x. This function is much more efficient than the natural MATLAB solution x(randperm(numel(x))

Syntax

shuffling(x)

Description

If x has t elements, the objective is to obtain each of the t! pemutations with equal probability, especially when t is large. To achieve this goal, we use Knuth shuffling in backward direction. Knuth solution, which is actually based on the Fisher-Yates-shuffle, is much more efficient than the natural MATLAB solution x(randperm(numel(x)).

    Note   if the vector x is big, it might be convenient to use parsimonious data type, e.g. shuffling(int8(1:20))

Example

>> v = [1 2 3 4 5];

>> shuffling(v)
