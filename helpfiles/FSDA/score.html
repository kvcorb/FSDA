Score

Approximate score test statistic for testing the transformation λ = λ0

Syntax

[outSC] = Score(y,X)
[outSC] = Score(y,X,param1,val1,param2,val2,...)

Description

[outSC] = Score(y,X) computes the approximate score statistic for testing the transformation λ = λ0 which is just the t-statistic for the coefficient of regression on a constructed variable which derives from first order Taylor expansion w(λ0) around λ0.

y is a vector of length n containing the response values, while X is the n-by-p-1 matrix which contains (in the columns) the values of p-1 predictive terms

[outSC] = Score(y,X) returns the following information

  • outSC − A structure of additional statistics with the following fields.

    • Score − length(la) x 1 vector containing the values of the score test for the supplied vector of transformation parameter

    • Lik − length(la) x 1 vector containing the values of the likelihood for the supplied vector of transformation parameter (if input value Lik =1)

      Note   Score automatically includes a constant term in all models. Do not enter a column of 1s directly into X.

[outSC] = Score(y,X,param1,val1,param2,val2,...) specifies one or more of the name/value pairs described in the following table.

Parameter Value
'intercept'

If 1, a model with constant term will be fitted (default) else no constant term will be included.

'la'

Vector which specifies for which values of the transformation parameter it is necessary to compute the score test. Default value of lambda is la=[-1 -0.5 0 0.5 1]; that is the five most common values of lambda
'Lik'

If Lik=1 the value of the likelihood for the augmented model will be produced else (default) only the value of the score test will be given
'nocheck'

Scalar. If nocheck is equal to 1 no check is performed on matrix y and matrix X. Notice that y and X are left unchanged. In other words the additional column of ones for the intercept is not added. As default nocheck=0.

Examples

Example 1

Compute the score test for the wool dataset:

XX=load('wool.txt')
y=XX(:,end);
X=XX(:,1:end-1);
% Compute the score test using the five most common values of the transformation
% parameter
[outSc]=Score(y,X)

outSc.Score =

   17.7059
    7.4927
   -0.9122
   -9.5511
  -18.5576

The constructed variable is not significant when we consider λ=0

Example 2

Compute the score test for the loyalty cards dataset:

load('loyalty.txt');
y=loyalty(:,4);
X=loyalty(:,1:3);
% la = vector containing the values of λ which have to be tested
la=[0.25 1/3 0.4 0.5];
[outSc]=Score(y,X,'la',la);
outSc =
    4.3160
   -0.2124
   -3.8334
   -9.4717

Using all the observations it seems that the third root is the best value of the transformation parameter.
