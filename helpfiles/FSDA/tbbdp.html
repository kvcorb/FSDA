TBbdp

Compute the value of the constant c which we have to put in  œÅ function for Tukey biweight to obtain a prefixed breakdown point

Syntax

c = TBbdp(bdp,p)

Description

c = TBbdp(bdp,p) calculates value of constant c which is necessary to obtain a certain breakdown point. bdp is a scalar containing desired breadkdown point. p is a scalar which denotes the number of response variables present in the dataset.

Examples

To obtain a nominal breakdown point of 0.5 in regression we have to se c equal to 1.547644980928226 
c=TBbdp(0.5,1)
