subsets

Creates a matrix of indexes where rows are distinct p-subsets extracted from a set of n elements

Syntax

[C] = subsets(nsamp,n,p)
[C] = subsets(nsamp,n,p,ncomb)
[C] = subsets(nsamp,n,p,ncomb,msg)
[C,nselected] = subsets(nsamp,n,p,ncomb,msg)

Description

C=subsets(nsamp,n,p) returns in matrix C the indexes of the distinct subsets which need to be extracted. Each row refers to a subset.

nsamp is a scalar which is associated with the number of subsamples which have be extracted. Remark: if nsamp=0 all subsets will be extracted. They will be (n choose p).

n is a scalar scalar, number of observations of the dataset.

p size of the subsets. In regression with p explanatory variable the size of the elmental subsets is p. In multivariate analysis in presente of v variables, the size of the elemental subsets is v+1

Optional arguments

ncomb ncomb = (n choose p) = n!/(p!(n-p)!). If the user has already computed this value it can supply it directly, otherwise the program will calculate it automatically

msg scalar which controls whether to display or not messages on the screen. If msg=1 (default) messages are displayed on the screen about estimated time.

The second output of this function (nselected) is nothing but the number ot rows of matrix C.

 

Examples

If we want to extract 120000 uniques subsets of size 5 from 100 elements we can use the following sintax

C=subsets(120000,100,5);
% we can verify that all the rows of matrix C are unique
size(unique(C,'rows'))
