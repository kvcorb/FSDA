htmlwriteFS

Is an obsolete function which will be removed in future releases. Use publishFS.m instead

Description

htmlwriteFS creates HTML files a MATLAB structure created with function mreadFS.m. Note that htmlwriteFS and mreadFS.m are obsolete and will be removed in future releases. USE publishFS.m instead.

More About

This function uses routine strjoin and can be used just by those which have at least MATLAB R2013a
