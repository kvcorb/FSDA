randsampleFS

Creates a matrix which contains, in the rows, all possible combinations of n elements taken k at time.

Syntax

randsampleFS(n,k)
y = randsampleFS(n,k)
y = randsampleFS(n,k,method)

Description

This function is an extension of randsample of the Statistics Toolbox.

y = randsample(n,k) returns a k-by-1 vector y of values sampled uniformly at random, without replacement, from the integers 1 to n.

 

method = scalar which determines the method used to extract the numbers.
if method=1 (default option) the program proceeds as follows:
if 4*k > n the programs does a random permutation of the population and returns the first nsel elements
else if 4*k <=n (that is if the desired sample is small compared to all combinations, the program repeatedly samples with replacement until there are nsel unique values.

if method=2 systematic sampling is used where the starting point is random and the step is also random

if method=3 random sampling based on the old but well known Linear Congruential Generator (LCG) method is used. In this case there is no guarantee to get unique numbers.

References (one for each method, in order):

Fisher, R.A.; Yates, F. (1948) [1938]. Statistical tables for biological, agricultural and medical research (3rd ed.). London: Oliver & Boyd. pp. 26–27.

Cochran, William G. (1977). Sampling techniques (Third ed.). Wiley.

D. E. Knuth. The Art of Computer Programming, Volume 2: Seminumerical Algorithms, Third Edition. Addison-Wesley, 1997. Section 3.2.1: The Linear Congruential Method, pp. 10–26.

Example

A random sample of 10 integers from the intergers 1:100. Default method 1 is used.

>> randsampleFS(100,10);
