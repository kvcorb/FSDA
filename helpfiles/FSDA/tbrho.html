TBrho

Compute ρ function for Tukey biweight

It transforms vector x as follows. Generally x is a vector containing residuals or Mahalanobis distances.

where c is a tuning constant which is linked to the breakdown point of the estimator.

Syntax

rhoTB = TBrho(u,c)

Description

y = TBrho(u,c) applies ρ function to elements of vector u. u is a n x 1 vector containing residuals or Mahalanobis distances for the n units of the sample and c scalar greater than 0 which controls the robustness/efficiency of the estimator

Examples

Graphical representation of bounded rho function for bisquare estimate 
x=-6:0.01:6;
rhoTB=TBrho(x,2);
plot(seq,rhoTB)
xlabel('x','Interpreter','Latex')
ylabel('$\rho (x)$','Interpreter','Latex')
